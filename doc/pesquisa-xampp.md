O que é o XAMPP

O XAMPP é um pacote de softwares utilizado para criar um ambiente de desenvolvimento web local. Ele permite executar aplicações PHP e utilizar banco de dados no próprio computador.

A sigla XAMPP significa:
X - Compatível com vários sistemas operacionais;
A -  Apache;
M -  MySQL/MariaDB;
P -  PHP;
P -  Perl.

Principais Componentes e Finalidade
Apache: Servidor web responsável por exibir páginas e aplicações no navegador.
MySQL/MariaDB: Sistema de gerenciamento de banco de dados utilizado para armazenar informações.
PHP: Linguagem de programação usada para criar aplicações web dinâmicas.
phpMyAdmin: Ferramenta gráfica utilizada para administrar bancos de dados MySQL/MariaDB.

Instalação e Configuração Básica
1 - Baixar o XAMPP no site oficial:
 https://www.apachefriends.org/
2 - Executar o instalador e selecionar os componentes desejados.
3 - Abrir o Painel de Controle do XAMPP.
4 - Iniciar os serviços:
Apache
MySQL

Importância do XAMPP para Desenvolvimento Local
O XAMPP é importante porque permite desenvolver e testar aplicações web localmente, sem precisar de hospedagem online. Ele facilita a integração entre servidor, PHP e banco de dados, tornando o desenvolvimento mais rápido, prático e organizado.
