Sistema Ferrorama - FJL Tech

Descrição do Projeto

O projeto Ferrorama tem como objetivo desenvolver um sistema web para gerenciamento de informações relacionadas ao controle e organização do sistema proposto pela equipe. A aplicação será desenvolvida utilizando tecnologias web, com foco em organização, usabilidade e aplicação de boas práticas de programação.


Objetivos do Projeto

Desenvolver um sistema organizado e funcional;
Aplicar conceitos de desenvolvimento web utilizando PHP;
Utilizar controle de versão com Git e GitHub;
Trabalhar em equipe utilizando divisão de tarefas;
Implementar operações CRUD no sistema;
Criar interfaces modernas e responsivas;
Aplicar boas práticas de estruturação e documentação.
